# CEventShooterController

## 설명

탄막 슈팅방의 슈터 컨트롤러, 슈터 이동속도, 슈터 최소최대 높이, 발사 주기, 발사 힘을 결정할 수 있다.

## Properties

| 변수명. | 설명. |
| :--- | :--- |
| shooterSpeed | 슈터 이동속도. |
| shooterMaxHeight | 슈터 최대 높이 |
| shooterMinHeight | 슈터 최소 높이 |
| firingCycle | 발사 주기 |
| launchVelocity | 발사 힘 |
