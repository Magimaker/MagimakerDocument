# CWindowFacade

## 설명 

인벤토리, 메뉴, 도움말 등 마우스 사용이 필요한 UI 창\(Window\)을 관리한다   
\(아이템 정보 등 실시간 팝업과 다름\)  
Window 사용 중에는 Cursor Lock이 중지되고 기존 컨트롤러 조작을 중단한다
